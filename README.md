# Haveno Website

Repository of the Haveno Website. Built on jekyll.

https://haveno.exchange

To test it locally:

- Install jekyll: https://jekyllrb.com/
- Run from root `bundle exec jekyll serve`
- Browse the website at http://127.0.0.1:4000/
